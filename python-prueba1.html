Nos descargarmos seaborn que tiene una interfaz para mostrar los datos, creamos una dataframe (df)

Importamos seaborn como sns para cargar los datos del archivo iris

Hacemos que el dataframe sea igual a la visualización de los datos de iris utilizando seaborn, previamente importado como sns

Hacemos que nos muestre el df con la linea de texto: sns.kdeplot (df["sepal_width"]) teniendo en cuenta la variable que queremos mostrar en la x del gráfico: sepal width

# library & dataset
import seaborn as sns
df = sns.load_dataset('iris')

# Make default density plot
sns.kdeplot(df['sepal_width'])
